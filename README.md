# llm-chatlog

## 개요

채팅로그 다운로드 크롬 확장프로그램 
chrome extension chatlog downloader and issue tracker

---

## 사용방법

- 크롬 개발자모드 on
- 압축해제된 확장프로그램 로드 클릭
- claude-chat-logger 폴더 업로드
- 저장하려고 하는 대화 페이지에서 F12 개발자모드 콘솔 열기
- ctrl+s 하면 다운로드 폴더에 저장됩니다.
